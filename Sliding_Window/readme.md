# Sliding Window

## Overview

Sliding Window is used to optimize subarray and substring problems.

## Time Complexities

| Approach | Complexity |
|-----------|-----------|
| Brute Force | O(n²) |
| Sliding Window | O(n) |

## Problems Covered

- Maximum Sum Subarray
- Longest Substring Without Repeating Characters
- Maximum Consecutive Ones
- Fruits Into Baskets
- Minimum Size Subarray Sum
- Longest Repeating Character Replacement
- Permutation in String

## Concepts Learned

- Fixed Size Window
- Variable Size Window
- Frequency Map
- Two Pointer Optimization
