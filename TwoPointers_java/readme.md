# Two Pointers

## Overview

Two Pointer technique reduces nested loops and improves efficiency.

## Time Complexities

| Approach | Complexity |
|-----------|-----------|
| Brute Force | O(n²) |
| Two Pointers | O(n) |

## Problems Covered

- Two Sum Sorted Array
- Remove Duplicates
- Move Zeroes
- Container With Most Water
- Trapping Rain Water
- 3Sum
- 4Sum
- Valid Palindrome

## Concepts Learned

- Opposite Direction Pointers
- Fast & Slow Pointer
- Cycle Detection
- Array Optimization
