# Binary Search

## Overview

Binary Search works on sorted data and reduces search time significantly.

## Time Complexities

| Operation | Complexity |
|------------|------------|
| Search | O(log n) |
| Insert | O(n) |
| Delete | O(n) |

## Problems Covered

- Binary Search
- First Occurrence
- Last Occurrence
- Search Insert Position
- Square Root
- Peak Element
- Search in Rotated Sorted Array
- Koko Eating Bananas
