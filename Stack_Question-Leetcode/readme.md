# Stack

## Overview

Stack follows the LIFO (Last In First Out) principle.

## Time Complexities

| Operation | Complexity |
|------------|------------|
| Push | O(1) |
| Pop | O(1) |
| Peek/Top | O(1) |
| Search | O(n) |
