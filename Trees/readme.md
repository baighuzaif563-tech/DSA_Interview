# Trees

## Time Complexities

| Operation | Complexity |
|------------|------------|
| DFS | O(n) |
| BFS | O(n) |
| Height | O(n) |
| BST Search | O(log n) Avg |
| BST Insert | O(log n) Avg |

## Problems Covered

- Inorder Traversal
- Preorder Traversal
- Postorder Traversal
- Level Order Traversal
- Lowest Common Ancestor
- Diameter of Tree
